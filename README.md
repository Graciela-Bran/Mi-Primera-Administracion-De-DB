<p>
  Presento una simple Base de Datos para un control escolar, esta tiene:
</p>

## Tablas que posee
- Estudiantes
- Grados
- Materias
- Profesores
- Asistencia

## Objetivo
<p>
El objetivo de esta base de datos es poder llevar un registro de los datos de un centro escolar, permitiendo registrar nuevos
estudiantes y mestros, mejorando el control que se tiene en la institucion y facilitando tareas los docentes.
</p>
